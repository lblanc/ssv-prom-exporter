S3 target (optional)

Configure an S3-compatible bucket to receive the .txt.gz files produced by Export. The "Push to S3" checkbox in Export only does something when this form is filled in. Credentials can be left empty for a publicly writable bucket (uncommon) or when relying on environment-supplied credentials.

Only used when "Bucket is publicly readable" is checked. If set, the share URL is built as <public_base_url>/<key>; otherwise it is <scheme>://<endpoint>/<bucket>/<key>.
